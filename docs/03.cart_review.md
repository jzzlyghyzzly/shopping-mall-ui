# 장바구니 페이지 리뷰

## 고객사 요구사항
-   **페이지 목적**: 사용자들이 여러 상품을 장바구니에 담아 한눈에 확인하고, 구매 결정을 내릴 수 있는 환경을 만드는 것이 목표입니다.
-   **표시 항목**: 상품 이미지, 브랜드명, 가격, 수량, 총 상품 금액, 배송비, 총 결제 금액을 명확하게 보여주도록 구현했습니다.
-   **배송비 정책**: 총 구매 금액이 10만원 이상일 경우, 배송비는 무료로 처리하고 이 사실을 사용자에게 명확히 알리도록 했습니다.
-   **수량 조절**: 각 상품 옆에 있는 `+`와 `-` 버튼을 눌러 상품의 수량을 조절할 수 있습니다.
-   **동적 정보 표시**: 페이지 상단에 장바구니에 담긴 상품이 총 몇 개인지 실시간으로 알려줍니다.
-   **페이지 이동**: '뒤로 가기' 버튼과 '결제하기' 버튼을 통해 각각 상품 목록 페이지와 결제 페이지로 이동할 수 있도록 연결했습니다.
-   **디자인 반영**: 제공해주신 시안의 레이아웃과 디자인을 최대한 비슷하게 구현했습니다. 특히, 상품 목록이 3개 이상일 경우 스크롤이 가능하도록 하여 여러 상품을 담을 수 있게 만들었습니다.

## 요구사항에 따른 기능 동작 확인

**장바구니 페이지의 주요 기능과 동작 상태를 고객의 입장에서 쉽게 설명해 드리겠습니다.**

1.  **상품이 담긴 장바구니 화면**:
    * 장바구니 페이지에 접속하면, 담겨 있는 상품들이 목록으로 나타납니다.
    * 화면 상단에는 "현재 n개의 상품이 담겨있습니다."와 같이 장바구니에 있는 총 상품 개수가 실시간으로 표시됩니다.
    * **동작 여부**: 문제없이 잘 동작합니다.

2.  **상품 수량 조절 및 삭제**:
    * 각 상품 옆에 있는 `+` 또는 `-` 버튼을 눌러 상품의 수량을 조절할 수 있습니다. 예를 들어, `+`를 누르면 수량이 1에서 2로 변하고, 총 결제 금액도 자동으로 다시 계산됩니다.
    * `-` 버튼을 계속 눌러 상품 수량을 0으로 만들면, 해당 상품은 장바구니 목록에서 자동으로 사라집니다.
    * **동작 여부**: 문제없이 잘 동작합니다.

3.  **배송비 및 금액 계산**:
    * 장바구니 하단에는 '상품 금액', '배송비', '총 금액'이 깔끔하게 정리되어 나타납니다.
    * **상품 금액**은 담겨 있는 모든 상품의 가격과 수량을 곱하여 자동으로 계산됩니다.
    * **배송비**는 총 상품 금액에 따라 달라집니다. 총 상품 금액이 10만원 미만이면 3,000원이 표시되지만, 10만원 이상이 되면 배송비가 '무료'로 바뀌고 최종 금액에 포함되지 않습니다.
    * **총 금액**은 상품 금액과 배송비를 합산한 최종 결제 금액을 보여줍니다.
    * **동작 여부**: 문제없이 잘 동작합니다.

4.  **페이지 이동**:
    * 페이지 왼쪽 상단의 `<` (뒤로 가기) 버튼을 누르면, 이전에 구현된 상품 목록 페이지로 다시 이동합니다.
    * 화면 하단의 '결제하기' 버튼을 누르면, 이전에 구현된 카드 선택 페이지로 이동하여 결제를 진행할 수 있습니다.
    * **동작 여부**: 문제없이 잘 동작합니다.

5.  **디자인 및 레이아웃**:
    * 제공해주신 시안을 바탕으로 장바구니 페이지의 전체적인 레이아웃을 구현했습니다. 다만, 모든 기기(PC, 모바일)에서 최적의 사용성을 제공하기 위해, 시안의 절대적인 위치 값보다는 유연하게 배치되는 반응형 디자인을 우선적으로 적용했습니다.
    * **동작 여부**: 잘 동작합니다.
