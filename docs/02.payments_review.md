# 결제 모듈 리뷰

## 고객사 요구사항
- 사용자 인터페이스: 제공된 피그마 시안과 동일한 디자인 및 레이아웃을 따릅니다.

- 반응형 디자인: 모바일 화면에 최적화되어 있으며, 태블릿 및 데스크톱 환경에서도 화면 크기에 맞춰 중앙에 정렬되는 유동적인 디자인을 적용했습니다.

- 카드 목록 및 등록:

    - 처음 접속 시, 등록된 카드가 없는 상태로 시작합니다.

    - 카드를 등록하면 목록에 추가되며, 여러 카드를 등록할 수 있습니다.

- 카드 번호 입력:

    - 16자리 숫자만 입력할 수 있으며, 입력 시 자동으로 'xxxx xxxx xxxx xxxx' 형식으로 공백이 추가됩니다.

- 만료일 입력:

    - MM/YY 형식으로 입력할 수 있으며, 유효하지 않은 월이나 지난 날짜는 입력이 제한됩니다.

- 카드 소유자 이름 입력:

    - 최대 30자까지 입력 가능하며, 입력하는 동안 (현재 글자 수 / 30)으로 실시간 글자 수를 표시합니다.

- 보안 코드 (CVC/CVV) 입력:

    - 정확히 3자리의 숫자만 입력 가능하도록 제한했습니다.

    - 입력 시 •••처럼 자동으로 가려집니다.

- 카드 비밀번호 입력:

    - 총 4자리 중, 첫 2자리만 사용자가 입력할 수 있으며, 나머지 2자리는 미리 •으로 가려진 상태로 표시됩니다.

- 입력 시 카드 미리보기:

    - 카드 번호, 소유자 이름, 만료일 정보를 입력할 때마다 화면 상단의 카드 이미지가 실시간으로 업데이트됩니다.

- 민감 정보 보안 처리:

    - 카드번호: 입력이 완료되면 뒤에서 8자리가 •••• •••• •••• ••••와 같이 가려집니다.

    - CVC/CVV 및 카드 비밀번호: 입력과 동시에 •으로 가려집니다.

- 버튼 동작:

    - 모든 필수 정보가 올바르게 입력되었을 때만 '작성 완료' 버튼이 활성화됩니다.

    - '이 카드로 결제하기' 버튼은 등록된 카드가 있을 때만 표시됩니다.

- 화면 이동:

    - 카드 추가 버튼 클릭 시 카드 등록 화면으로 이동합니다.

    - 카드 등록 후 '작성 완료' 버튼 클릭 시, 등록된 카드가 추가된 상태로 보유카드 화면에 다시 나타납니다.

    - '뒤로 가기' 및 '닫기' 버튼은 각 화면의 목적에 맞게 정상적으로 동작합니다.

## 요구사항에 따른 기능 동작 확인
사용자 경험을 중심으로 구현된 기능의 동작 방식을 설명합니다.

- 앱 시작 시 첫 화면:

    - 앱을 처음 시작하면, 사용자가 등록한 카드가 없는 빈 화면이 나타납니다. 화면 중앙에는 "새로운 카드를 등록해주세요."라는 문구와 함께 카드 모양의 버튼이 있습니다.

    - 이 버튼을 누르면 카드를 등록하는 화면으로 자연스럽게 넘어갑니다.

- 카드 정보 입력 화면:

    - 이 화면에서는 화면 상단에 카드 미리보기가 표시됩니다.

    - 카드 번호: 16자리를 입력하면, 자동으로 4자리마다 공백이 생겨 '1234 5678 ...'과 같이 보기 좋게 정리됩니다. 입력이 끝나면 카드 미리보기의 번호가 뒷부분만 가려진 채로 바뀝니다.

    - 만료일: 'MM/YY' 형식에 맞춰 숫자를 입력하면 자동으로 '/'가 표시됩니다. 유효하지 않은 월이나 이미 지난 연도는 입력이 되지 않습니다.

    - 카드 소유자 이름: 이름을 입력할 때마다 아래에 '1/30'과 같이 현재 글자 수가 표시됩니다. 30자를 넘기면 더 이상 입력할 수 없습니다.

    - 보안 코드 (CVC): 3자리의 숫자를 입력할 수 있으며, 입력 즉시 •으로 가려집니다.

    - 카드 비밀번호: 첫 두 자리만 입력할 수 있고, 입력하는 숫자는 •으로 가려집니다. 나머지 두 자리는 입력 없이 미리 •으로 표시됩니다.

    - 버튼 활성화: 모든 입력 칸에 필수 정보가 올바르게 채워져야만 화면 아래의 '작성 완료' 버튼이 노란색으로 바뀌며 활성화됩니다.

- 카드 등록 완료:

    - '작성 완료' 버튼을 누르면, '카드 등록 완료!' 알림이 뜬 후 첫 화면으로 돌아갑니다.

    - 이때 첫 화면에는 방금 입력했던 카드가 추가되어 있습니다. 카드 번호와 만료일, 소유자 이름이 시안과 같이 표시됩니다.

    - 카드 아래에는 "이 카드로 결제하기" 버튼이 활성화되어 있으며, 그 아래에는 "다른 카드 추가" 버튼이 표시됩니다.

    - '다른 카드 추가' 버튼을 누르면 다시 카드 정보 입력 화면으로 넘어갈 수 있으며, 추가한 카드는 목록의 가장 위에 추가됩니다.
